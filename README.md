## ✨ 项目介绍

这是一个开源的前端开发学习资源库，包含了前端开发的核心知识点、最佳实践和实战经验。通过系统化的内容组织，帮助开发者建立完整的前端知识体系。

### 特色亮点

- 📖 **系统化学习路径** - 从 HTML/CSS 基础到框架应用，循序渐进
- 🧭 **实用知识体系** - 覆盖前端开发的全方位内容
- ⚡ **高效导航** - 基于 VitePress 的现代化文档体验
- 📱 **响应式设计** - 完美适配各种设备
- 🎯 **持续更新** - 定期补充新技术和最佳实践

## 🚀 快速开始

### 前置要求

- Node.js >= 16.0.0
- npm 或 yarn

### 本地开发

```bash
# 克隆项目
git clone https://github.com/he-zhiyuan/fedev-wiki.git
cd fedev-wiki

# 安装依赖
npm install

# 启动开发服务器
npm run docs:dev

# 打开浏览器访问 http://localhost:5173/
```

### 构建生产版本

```bash
# 构建静态网站
npm run docs:build

# 预览生产构建
npm run docs:preview
```

## 📊 学习建议
