# Vue.js 框架

Vue.js 是一个流行的 JavaScript 前端框架，用于构建用户界面和单页应用程序。本部分内容涵盖了 Vue.js 的核心概念和实践应用。

## 学习路径

1. [模板语法与指令系统](/vue/template) - 学习 Vue 的模板语法和内置指令
2. [响应式系统与 Ref/Reactive](/vue/reactive) - 了解 Vue 的响应式原理
3. [组件化开发](/vue/components) - 掌握组件的创建和组合
4. [生命周期钩子](/vue/lifecycle) - 理解组件的生命周期
5. [路由与动态路由](/vue/router) - 使用 Vue Router 构建多页面应用
6. [状态管理（Pinia）](/vue/pinia) - 学习现代状态管理方案
7. [Composition API 与 Setup 语法糖](/vue/composition) - 掌握组合式 API
8. [Vue DevTools 与调试技巧](/vue/devtools) - 提高开发和调试效率

## Vue.js 概览

Vue.js 是一个渐进式框架，它允许开发者：

- 逐步采用框架特性
- 构建响应式的用户界面
- 创建可复用的组件
- 管理应用状态
- 处理客户端路由
- 轻松集成第三方库

Vue.js 以其简单的 API 和出色的性能而闻名，是现代前端开发的热门选择。
