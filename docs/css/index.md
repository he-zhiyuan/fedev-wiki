# CSS 基础

CSS（层叠样式表）用于控制网页的外观和布局。本部分内容涵盖了从基础到高级的CSS知识。

## 学习路径

1. [选择器](/css/selectors) - 了解如何选择和定位HTML元素
2. [盒模型](/css/box-model) - 掌握CSS的核心概念：盒模型
3. [定位](/css/position) - 学习不同的定位方法
4. [浮动](/css/float) - 理解浮动布局和清除浮动
5. [Flex 布局](/css/flex) - 掌握弹性盒子布局
6. [Grid 布局](/css/grid) - 学习网格布局系统
7. [媒体查询](/css/media-queries) - 创建响应式设计
8. [动画](/css/animation) - 实现CSS动画和过渡效果
9. [SASS/SCSS 基础](/css/sass) - 使用CSS预处理器提高效率
10. [层叠与继承](/css/cascade) - 深入理解CSS的核心机制
11. [BEM 命名规范](/css/bem) - 学习模块化CSS命名方法

## CSS 概览

CSS是网页设计的核心技术之一，它允许开发者：

- 控制文本样式（字体、大小、颜色等）
- 创建布局和定位元素
- 添加动画和过渡效果
- 实现响应式设计
- 自定义用户界面组件外观

掌握CSS是成为优秀前端开发者的关键步骤。
