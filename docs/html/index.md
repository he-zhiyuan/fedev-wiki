# HTML 基础

HTML（超文本标记语言）是构建网页的基础。本部分内容涵盖了HTML的各个方面，从基本结构到高级特性。

## 学习路径

1. [基本结构](/html/structure) - 学习HTML文档的基本结构和元素
2. [常用标签](/html/tags) - 了解最常用的HTML标签及其用法
3. [表单控件](/html/forms) - 掌握表单元素和数据提交
4. [语义化标签](/html/semantic) - 使用语义化标签提高代码可读性和可访问性
5. [媒体标签](/html/media) - 在网页中嵌入音频、视频和图像
6. [可访问性](/html/accessibility) - 创建对所有用户友好的网页

## HTML 概览

HTML是一种用于创建网页的标准标记语言。通过HTML，可以定义网页内容的结构，包括：

- 文本内容
- 多媒体内容
- 超链接
- 表单
- 表格
- 列表
- 和许多其他内容类型

良好的HTML基础是掌握前端开发的第一步，也是最重要的步骤之一。
